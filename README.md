# 🏠 House Price Prediction

This project uses machine learning techniques to predict house prices based on housing data. The model is trained using real-world dataset features and implemented using Python.

---

## 📊 Dataset

The dataset (`housing.csv`) contains housing-related features used to train the model, such as property size and other attributes that influence house prices.

---

## 🚀 Technologies Used

- Python  
- Pandas  
- NumPy  
- Scikit-learn  
- Jupyter Notebook  

---

## ⚙️ Project Workflow

1. Data Loading  
2. Data Exploration and Cleaning  
3. Feature Selection  
4. Model Training (Linear Regression)  
5. Model Evaluation  
6. Price Prediction  

---

## 📁 Project Structure
house-price-prediction/
│
├── housing.csv # Dataset
├── housing.ipynb # Jupyter Notebook (model + analysis)
├── README.md # Project documentation


---

## 📈 Model

The model used in this project is **Linear Regression**, a supervised machine learning algorithm used for predicting continuous values.

---

## ▶️ How to Run

1. Install dependencies:pip install pandas numpy scikit-learn
2. Open the notebook:jupyter notebook housing.ipynb
3. Run all cells to see predictions

---

## 🎯 Results

The model is trained on housing data and can predict house prices based on input features.
